Loc is a web tool that loads a folder from your local file system and tabulates files with their path and line count.

It is possible to fine-tune the considered files using glob pattern syntax. You can easily copy and paste the computed data on your favorite spreadsheet software to further your analysis and estimates.

The app uses the secure File System API and does not send data from your machine. Everything is done right in your browser. The source code is available in this GitHub repository. You can check it or even download and run it locally. The site is hosted in the same infrastructure using GitHub Pages.
